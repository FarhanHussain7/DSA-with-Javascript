Loop Through Object Properties - for...in, forEach, entries

âœ… for...in Loop
Loop through all enumerable properties of an object (including inherited ones). 
const person = {
    name: "John",
    age: 30,
    email: "john@example.com",
    city: "New York"
};

// Basic for...in loop
for (let key in person) {
    console.log(key + ": " + person[key]);
}

// Output:
// name: John
// age: 30
// email: john@example.com
// city: New York

// With prototype
const proto = { country: "USA" };
const user = Object.create(proto);
user.name = "Alice";
user.age = 25;

for (let key in user) {
    console.log(key); // name, age, country (includes inherited)
}

// Filter out inherited properties
for (let key in user) {
    if (user.hasOwnProperty(key)) {
        console.log(key); // only name, age (own properties)
    }
}
âœ… Object.keys().forEach()
Loop through enumerable own properties using forEach with keys. 
const person = {
    name: "John",
    age: 30,
    email: "john@example.com"
};

// Method 1: Simple forEach
Object.keys(person).forEach(key => {
    console.log(key + ": " + person[key]);
});

// Method 2: With index
Object.keys(person).forEach((key, index) => {
    console.log(index + 1 + ". " + key + ": " + person[key]);
});

// Method 3: Traditional for loop alternative
const keys = Object.keys(person);
for (let i = 0; i < keys.length; i++) {
    const key = keys[i];
    console.log(key + ": " + person[key]);
}
âœ… Object.values().forEach()
Loop through object values directly without needing keys. 
const person = {
    name: "John",
    age: 30,
    email: "john@example.com",
    city: "New York"
};

// Iterate over values only
Object.values(person).forEach(value => {
    console.log(value);
});

// Output:
// John
// 30
// john@example.com
// New York

// With type checking
Object.values(person).forEach(value => {
    if (typeof value === "number") {
        console.log(value + " (number)");
    } else if (typeof value === "string") {
        console.log(value + " (string)");
    }
});
âœ… Object.entries().forEach() - Best Method
Loop through key-value pairs. Recommended method for iteration. 
const person = {
    name: "John",
    age: 30,
    email: "john@example.com",
    city: "New York"
};

// Destructured entries
Object.entries(person).forEach(([key, value]) => {
    console.log(key + ": " + value);
});

// With index
Object.entries(person).forEach(([key, value], index) => {
    console.log((index + 1) + ". " + key + ": " + value);
});

// Without destructuring
Object.entries(person).forEach(entry => {
    console.log(entry[0] + " = " + entry[1]);
});

// Filter while iterating
Object.entries(person).forEach(([key, value]) => {
    if (typeof value === "string") {
        console.log("String: " + key + " = " + value);
    }
});
âœ… Object to Map Conversion
Convert object to Map for advanced iteration patterns. 
const person = {
    name: "John",
    age: 30,
    email: "john@example.com"
};

// Convert object to Map
const personMap = new Map(Object.entries(person));

console.log(personMap);
// Map { "name" => "John", "age" => 30, "email" => "john@example.com" }

// Iterate Map entries
personMap.forEach((value, key) => {
    console.log(key + ": " + value);
});

// Get Map size
console.log("Total properties: " + personMap.size);

// Check if key exists
console.log(personMap.has("name")); // true
console.log(personMap.has("phone")); // false

// Get value from Map
console.log(personMap.get("age")); // 30

// Convert Map back to Object
const restored = Object.fromEntries(personMap);
console.log(restored);
// { name: "John", age: 30, email: "john@example.com" }
