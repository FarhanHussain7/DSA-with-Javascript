A linked list is a linear data structure where elements are not stored at contiguous memory locations. Each element (node) contains:

  • Data: The actual value stored
  • Next Pointer: Reference to the next node

Node Class Implementation

class Node { constructor(data) { this.data = data; // Store the value this.next = null; // Pointer to next node (initially null) } } // Example usage: const node1 = new Node(10); const node2 = new Node(20); const node3 = new Node(30); // Link the nodes together node1.next = node2; node2.next = node3; console.log(node1.data); // Output: 10 console.log(node1.next.data); // Output: 20 console.log(node1.next.next.data); // Output: 30

LinkedList Class Implementation

class LinkedList { constructor() { this.head = null; // First node (initially null) this.size = 0; // Track number of nodes } // Check if list is empty isEmpty() { return this.size === 0; } // Get current size getSize() { return this.size; } }

Basic Operations

// Add first node to empty list addFirst(data) { const newNode = new Node(data); this.head = newNode; this.size = 1; } // Display list contents display() { if (this.isEmpty()) { return "List is empty"; } let current = this.head; let result = ''; while (current !== null) { result += current.data; if (current.next !== null) { result += ' -> '; } current = current.next; } return result; } // Get first element getFirst() { if (this.isEmpty()) { return null; } return this.head.data; } // Clear the list clear() { this.head = null; this.size = 0; }
